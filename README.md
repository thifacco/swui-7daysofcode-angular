# Star Wars UI

Projeto desenvolvido com Angular v15.2.0-lte para treinar consumo de API [SWAPI](https://swapi.dev) proposto pelo desafio #7DaysOfCode da Alura.

### Features

- Home: página principal da aplicação
- Movies: lista todos os filmes da franquia Star Wars
- Ships: Lista todas as naves da franquia Star Wars

### Running app

```
npm start
```

## Running unit tests

```
npm run test
```
